# mdgrep

This is the fastest grep tool on OSX!

By taking advantage of Spotlight index via `mdfind`, piping through `grep`!

## Installation

Clone this tool and copy `mdgrep` to your `/usr/local/bin`

## Usage

```
mdgrep 'search string'
```
